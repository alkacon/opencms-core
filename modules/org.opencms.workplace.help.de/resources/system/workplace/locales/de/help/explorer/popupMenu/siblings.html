Hier können Sie siche alle Verknüpfungen, die auf den gleichen Inhalt zeigen (Geschwister) anzeigen lassen.  Wenn sie den Inhalt einer Datei verändern, zu der es mehrere Verknüpfungen gibt, so wird "der Inhalt aller Verknüpfungen" geändert. D.h., dass alle Vernküpfungen auf den gleichen Inhalt zeigen.

Mehr über das Thema Verknüpfungen erfahren Sie hier.

Bitte beachten Sie: Diese Aktion kann nur für Dateien ausgeführt werden, die mehrere Verknüpfungen aufweisen. Wenn dies nicht der Fall ist, erscheint der Menüpunkt des Kontextmenüs ausgegraut.
