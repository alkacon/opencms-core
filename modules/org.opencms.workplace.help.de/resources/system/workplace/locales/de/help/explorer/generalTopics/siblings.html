Als Verknüpfungen werden inhaltlich identische Seiten und Dateien bezeichnet. Eine neue Verknüpfung mit einer Ressource können Sie über den Dialog "Neu" in der Titelleiste erstellen. Das Ziel einer Verknüpfung kann auch in anderen Verzeichnissen liegen.  Wenn Sie eine verknüpfte Seite bearbeiten, werden damit auf dieselbe Weise auch alle anderen verknüpften Seiten geändert. Sie können so den Inhalt einer Seite in unterschiedlichen Verzeichnissen als „Verknüpfung“ ablegen und brauchen nur eine Datei bearbeiten um sämtliche verknüpfte Dateien ebenfalls zu ändern. Dies hat dann Vorteile, wenn ein bestimmter Inhalt auf einer Seite in unterschiedlichen Themenbereichen und  Zusammenhängen als eigenständige Seite präsentiert werden soll.

Beispiel: Es müssen die Allgemeinen Geschäftsbedingungen (AGB) in den Internetauftritten von allen angeschlossenen Filialen erscheinen. Der Inhalt soll aber zentral von einer autorisierten Person gepflegt werden. Ein Link auf die „AGB“ zur Geschäftsführung wäre unangemessen, weil der Besucher im Internetauftritt der jeweiligen Filiale verbleiben soll. Also wird die Seite der „AGB“ als Verknüpfung in die Verzeichnisse der einzelnen Filialen kopiert und kann im Verzeichnis der autorisierten Person bearbeitet werden.

Solange mehr als eine Seite, die auf einen Inhalt zeigt existiert, sind die Seiten Verknüpfungen. Werden alle bis auf eine Verknüpfung zum gleichen Inhalt gelöscht, dann wird die letzte Verknüpfung zu dem Original und fortlaufend normal dargestellt und behandelt.

Achtung: Wenn sie Verknüpfungen löschen sollten Sie die Option "Verknüpfungen erhalten " auswählen, um nicht alle Verknüpfungen (und damit den Inhalt) gleichzeitig zu löschen. Es sei denn, Sie beabsichtigen dies.

Achtung: Es können keine Verknüpfungen auf Ordner erstellt werden. Wird ein Ordner als Verknüpfung abgelegt, so ist der Ordner eine eigenständige Kopie. Nur die Inhalte des Ordners haben dann verknüpfte Elemente mit den oben beschriebenen Eigenschaften. Wird in einem der so kopierten Ordner eine neue Seite angelegt oder neue Dateien hochgeladen, wirkt sich das nicht auf den anderen Ordner aus.

Tip: Sie können Verknüpfungen durch die Schaltfläche "Neu" oder durch Kopieren von Dateien mit Auswahl "Erstelle eine neue Verknüpfung zur Datei" erstellen.
