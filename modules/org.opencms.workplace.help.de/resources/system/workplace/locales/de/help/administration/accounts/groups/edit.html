Um eine Gruppe für OpenCms zu erstellen oder zu bearbeiten stehen Ihnen für alle erforderlichen Eingaben als Eingabefelder zur Verfügung:

  • Name

Alle restlichen Felder sind optional:

  • Beschreibung:
    Hier können sie weitere Informationen über die Bedeutung der Gruppe angeben.
  • Obergruppe:
    Von dieser Gruppe werden Berechtigungen geerbt. Alles was die Obergruppe darf, darf auch diese Gruppe - ebenso verhält es sich mit den Verboten.
  • Gruppe als Rolle:
    Wenn aktiviert, wird die Gruppe zusätzlich auch als Rolle verwedbar.
  • Projektmanager Gruppe:
    Die Gruppe erhält die Standardberechtigungen, die für einen Projectmanager erforderlich sind.
  • Projekt Mitarbeiter:
    Die Gurppe erhält die Standardberechtigungen, die für einen Projektmitarbeiter erforderlich sind.
