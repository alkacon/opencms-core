Mit Hilfe dieses Menüpunktes erstellen Sie eine Verknüpfung auf externe Inhalte, sprich auf Webseiten die außerhalb Ihres Webservers liegen. Nachdem Sie im Dialogfenster „ Neu“ die Auswahl „Externe Verknüpfung“ gewählt haben, wird der Dialog entsprechend fortgeführt. In zunächst zwei weiteren Dialogfenstern werden alle relevanten Parameter abgefragt.

  • Name       
    Vergeben Sie einen eindeutigen und verständlichen Namen für die neue Datei. Beachten Sie, dass der Dateiname weder Leerzeichen, noch Sonderzeichen (ä, ü, ö, ß....) enthält. Dies hat grundsätzlich eine Fehlermeldung zur Folge. Achten Sie auf eine durchgängige Kleinschreibung.
  • Link URL 
     Geben Sie hier die URL der zu verlinkenden Seite an. Vergessen Sie nicht das http:// voranzustellen.
  • Sofern die Eigenschaft „Bearbeite Eigenschaften der neuen Datei“ aktiviert wurde, wird der Dialogverlauf entsprechend um zwei weiteren Eingabefenstern erweitert In diesen können die Eigenschaften der zu erstellenden Datei vergeben werden
  • Dialogfenster: Eigenschaften  
    Füllen Sie in diesem Dialogfenster alle wichtigen Eingabefelder aus. Bestätigen Sie anschließend mit einem Klick auf „ Fertig stellen“.

 

Tip: Sie können durch Einstellung der Eigenschaften externe Verknüfpungen transparent in Ihre Seitennavigation einbauen.
