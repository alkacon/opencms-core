Es gibt verschiedene Wege Web - Inhalte in OpenCms zu erstellen und editieren: WYSIWIG ("What you see is what you get") Editoren - also Editoren wie z.B. Microsoft Word, bei denen Sie genau das, was Sie auf dem Bildschirm editieren auch als Dokument herausgeben. WYSIWIG - Editoren erlauben einfaches Editieren von HTML - Seiten ("Hypertext Markup Language", Das Format von Dokumenten im Internet). D.h. Sie produzieren mit Ihnen Webseiten, die HTML - code enthalten ohne diesen Code selbst zu sehen oder gar verstehen zu müssen.

Die beiden WYSIWYG - Editoren von OpenCms sind:;

  • MSDHTML - der Microsoft Dynamic Html für den Internet Explorer
  • HTMLArea - der Editor für alle Browser
