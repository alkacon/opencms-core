Das Berechtigungssystem in OpenCms 6.0.0 basiert auf sogenannten Access Control Lists. Jeder Ressource kann für jeden Benutzer oder jede Gruppe eine Liste von Zugriffsrechten zugeordnet werden, die aussagt, welche Arten von Zugriff  der Benutzer oder die Gruppe zu dieser Resource hat. Somit kann flexibel jede Ressource für jeden Benutzer oder jede Gruppe (die Benutzer enthält) die unterschiedlichen Arten des Zugriffs kontrolliert werden.

Weiterhin können dies Zugriffslisten auf Wunsch von Verzeichnissen an Unterverzeichnisse und enthaltene Dateien vererbt werden.

In diesem Abschnitt werden die folgenden Konzepte vorgestellt:

  • Arten des Zugriffs
  • Vererbung von ACL's
