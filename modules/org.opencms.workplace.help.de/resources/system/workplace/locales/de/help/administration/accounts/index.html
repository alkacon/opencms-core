Diese Seite erlaubt das Bearbeiten von Benutzern sowie Gruppen. Das Sicherheitsmodell von OpenCms 6.0 basiert auf Benutzern, Gruppen und Ressourcen / Tools. Eine Ressource (z.B. eine Seite oder eine andere Datei) gewährt nur bestimmten Gruppen Zugriff. Ein Tool (Werkzeug, dieser Begriff repräsentiert eine Aktion, die der Benutzer in der Workplace ausführen will) verlangt ebenso Gruppenzugehörigkeit, bevor ein Zugang stattgegeben wird.   

Auf der anderen Seite kann ein Benutzer Mitglied in Gruppen sein. Um Zugriff auf Tools oder Resourcen zu bekommen, muss ein Benutzer das Mitglied der Gruppe sein,  die die Resource oder das Tool zwecks Zugriff verlangt.

Weiterhin werden Benutzterdaten auch für weitere Aufgaben wie z.B. das Versenden von Email- oder Broadcast- Nachrichten (im "Workplace Werzeuge" Dialog) verwendet.
