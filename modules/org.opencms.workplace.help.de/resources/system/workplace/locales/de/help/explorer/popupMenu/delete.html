Löscht eine Datei oder ein Verzeichnis aus dem Projekt. Nach dem Löschvorgang erscheint das Element durchgestrichen im Workplace und wird erst nach einem Veröffentlichen vollständig aus Ihrem Projekt entfernt.

Ein gelöschtes (und damit durchgestrichen dargestelltes) Element können Sie wiederherstellen, indem Sie nach einem Klick auf das Datei- oder Verzeichnissymbol den Menüeintrag „ Wiederherstellen“ auswählen.

Bitte beachten Sie: Wenn sie eine Verknüpfung löschen, bekommen sie die Möglichkeit, alle Verknüpfungen zu löschen, zur Auswahl. Damit würden Sie den Inhalt und alle Seiten, die auf Ihn verweisen löschen. Wählen Sie die andere Option (als Standard aktiviert), um nur diese eine Verknüpfung zu löschen.

Erweiterte Optionen beim Löschen einer Verknüpfung.

Bitte beachten Sie: Sie können den Löschvorgang nur rückgängig machen, wenn die Datei Sie nach dem Löschen (oder das zug. Projekt) nicht veröffentlicht wurde.
