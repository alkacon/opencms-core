A dynamic function consists of two main parts: the function itself and a function provider, that is kind of a formatter for the function.

The function itself is an XML content where the provider, element settings and parameters passed to the function provider are configured. ]]> <![CDATA[Динамическая функция]]> Динамическая функция состоит из двух основных частей: сама функция и функция поставщика, то есть своего рода форматтер для функции.

Сама функция представляет собой XML-контент, где настраиваются форматтер, параметры элементов и параметры, передаваемые форматтером функции. Кроме того, могут быть определены контейнеры, в которые можно добавить функцию (возможно, с различными форматтерами).

Второй компонент, форматтер функции, это JSP-страница, которая отвечает за отображение содержимого и, следовательно, может считывать параметры и настройки элемента из функции.
