Dies ist ein einfaches Beispiel für einen mehrsprachigen Text unter Verwendung der neuen Verknüpfungen Funktion von OpenCms 6.0. Mittels Verknüpfungen kann ein Inhalt über mehrere URIs des OpenCms VFS erreichbar gemacht werden.

Mittels der locale Eigenschaft jeder Datei (oder eines übergeordneten Ordners) können Sie kontrollieren welche Sprache beim Aufruf der Seite angezeigt wird.

Um herauszufinden wie OpenCms 6.0 den Inhalt einer solchen Datei verwaltet, wählen Sie die Option  Steuerdatei bearbeiten im Kontextmenü dieser Datei in der OpenCms Workplace. Sie werden eine einfach zu verstehende XML Struktur vorfinden.

Die verfügbaren Sprachen (locales), ebenso wie die Vorgabesprache, wird in der OpenCms Konfigurationsdatei opencms-system.xml eingestellt.

This is a simple example for a multilingual text using the new VFS sibling feature of OpenCms 6.0. A sibling is a content that has more than one URI in the OpenCms VFS.

Through the locale property on each file (or a parent folder) you can control which locale is used when the file is requested by a user.

To check out how OpenCms 6.0 organizes the file content, you can select Edit controlcode in the context menu of this file in the OpenCms Workplace. You will find an easy to understand XML structure.

The available locales as well as the default locale are configured in the OpenCms configuration file opencms-system.xml.
