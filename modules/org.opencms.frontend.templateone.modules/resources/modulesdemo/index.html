Modules Demo pages

These pages show the usage of XML Contents with the following content types available for template one:

  • Events
  • Frequently asked questions (FAQ)
  • Job offerings
  • News messages

For these four types, variable lists of items can be displayed either on the right side of the template or in the content area below editable elements.

Those lists are activated or deactivated by choosing a layout with the "simplified" property editor that comes with the templateone module. Simply click on "Properties" in the context menu of a page or a folder using template one, the dialog allows to select a layout for the center and the right area of the template.

Note: To display XML Contents correctly, each microsite using templateone needs a "modules/" subfolder containing folders and items for the contents to display.
