Your personal settings can be edited by clicking on the button "preferences".

The Preferences-Button

"Workplace"-Tab

This page contains the the OpenCms start settings as your language, the site and folder you want to start with. Further, you can select whether you want to use applet to upload files, which is useful when uploading several files at once.

"User Data"-Tab

Here, you can check your user data and change your password.
