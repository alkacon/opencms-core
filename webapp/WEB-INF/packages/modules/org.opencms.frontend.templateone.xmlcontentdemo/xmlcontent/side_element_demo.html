This page demonstrates a list of editable XML content elements on the right side

Using XML content, you are able to quickly create and edit such lists of structured content. Creating template elements like the list on the right side is fairly easy and requires only a little OpenCms template programming knowledge.

Using the templates properties, the contents of the right side can be controlled. In this example, the configuration is done using the xmlcontent-side-uri property.
